Création Vault Chiffré pour Stockage Sécurisé

Interface VeraCrypt - Création de volume chiffré

  1. Téléchargement : Installer VeraCrypt depuis le site officiel
  2. Création volume : Sélectionnez "Create Volume" > "Create an encrypted file container"
  3. Emplacement : Choisir le dossier de la VM : E:\SecureCompartments\[ID]\encrypted_vault.hc
  4. Algorithme : AES-256 + SHA-512 (maximum sécurité)
  5. Taille : 10-20 GB selon besoins
  6. Mot de passe : Minimum 20 caractères avec complexité élevée
  7. Format : NTFS pour compatibilité Windows

            threading.Thread(target=self.monitor_system_resources, daemon=True),
