Glossaire - Termes que Vous Devez Connaître

Google Colab

Un "carnet de programmation" gratuit de Google qui fonctionne dans votre navigateur. C'est là que nous allons écrire et exécuter du code pour créer votre IA.

Stable Diffusion

Le "cerveau" de l'intelligence artificielle qui génère des images. C'est un programme informatique capable de créer des photos réalistes à partir de descriptions.

Dataset

Une collection d'images (comme un album photo) que nous montrons à l'IA pour lui apprendre à reproduire un visage spécifique.

Prompt

La "commande" que vous donnez à l'IA pour lui dire quel type d'image générer. Par exemple : "une femme brune souriante".

WebUI

L'interface graphique (écran avec boutons et menus) qui vous permet de contrôler l'IA sans écrire de code.

NSFW

"Not Safe For Work" - Contenu pour adultes, explicite ou nu. Nous l'utilisons de manière légale et transparente.

BLOC 2 - Création du Dataset Training

Objectif et Stratégie du Dataset

Qu'est-ce qu'un Dataset et Pourquoi en Avoir Besoin ?

Un dataset est comme un "album photo" que nous montrons à l'intelligence artificielle pour lui apprendre à reproduire un visage spécifique. Voici la logique :

Apprentissage IA

L'IA analyse 50-100 photos du même visage pour comprendre ses caractéristiques uniques : forme des yeux, nez, bouche, expressions, etc.

Reproduction

Une fois entraînée, l'IA peut générer de nouvelles images du même personnage dans des poses, tenues, situations différentes.

Consistance

Le même visage, reconnaissable, sur des milliers d'images différentes pour créer du contenu cohérent.

Spécifications Techniques du Dataset

Pour un entraînement optimal, votre dataset doit respecter ces critères précis :

Quantité et Qualité
  • • 50 à 100 images total
  • • Résolution minimum : 512x512 pixels
  • • Format : JPG ou PNG
  • • Qualité : nette, bien éclairée
  • • Pas de flou, déformation, coupure
Diversité Requise
  • • Expressions : souriante, sérieuse, séductrice
  • • Angles : face, profil, 3/4
  • • Cadrages : portrait, buste, corps entier
  • • Tenues : habillée, lingerie, nue
  • • Éclairages : naturel, studio, intime

Génération du Dataset - Méthode Professionnelle

Stratégie de Création du Personnage

Nous allons créer un personnage fictif cohérent. Voici la méthode professionnelle :

Phase 1 : Visage de Base
  • • Générer 10 variations d'un même prompt
  • • Choisir le visage le plus cohérent
  • • Noter le seed pour la consistance
Phase 2 : Variations SFW
  • • 30 images habillées diverses
  • • Expressions et poses variées
  • • Angles et cadrages différents
Phase 3 : Contenu NSFW
  • • 30 images lingerie/nude
  • • Progression graduelle
  • • Anatomie parfaite requise

Phase 1 : Création du Visage de Base

1
Premier Prompt de Test

Nous commençons par un prompt simple pour établir les bases du visage :

Prompt à Copier dans l'Interface :
portrait of a beautiful 25-year-old woman, european features, long brown hair, green eyes, natural makeup, soft lighting, photorealistic, detailed skin texture, 8k, high quality, professional photography
Negative Prompt (Éléments à Éviter) :
ugly, deformed, blurry, bad anatomy, extra limbs, missing limbs, bad hands, bad fingers, watermark, signature, text, logo, low quality, worst quality, jpeg artifacts
  • • Copiez le prompt dans la boîte "Prompt" (grande zone de texte)
  • • Copiez le negative prompt dans "Negative prompt"
  • • NE CLIQUEZ PAS ENCORE sur Generate
2
Configuration des Paramètres Optimaux

Réglez ces paramètres pour une qualité maximale :

Paramètres de Base :
  • • Sampling Steps : 30
  • • CFG Scale : 7
  • • Width : 512
  • • Height : 768
Paramètres Avancés :
  • • Sampling method : DPM++ 2M Karras
  • • Batch count : 1
  • • Batch size : 1
  • • Seed : -1 (aléatoire)
Où Trouver Ces Paramètres :
  • • Sous les boîtes de prompt, vous voyez des curseurs et menus
  • • "Width" et "Height" sont côte à côte
  • • "CFG Scale" est un curseur gradué 1-20
  • • "Sampling method" est un menu déroulant

CAPTURE D'ÉCRAN À AJOUTER :
Zone des paramètres avec tous les réglages mentionnés encadrés et annotés

3
Première Génération
  • • Une fois tous les paramètres réglés, cliquez sur le bouton "Generate" (orange)
  • • Une barre de progression apparaît
  • • Attendez 30-60 secondes (selon le GPU disponible)
  • • L'image apparaît dans la zone de droite
Résultat Attendu :

Vous devriez obtenir une image de femme ressemblant à ceci :

  • • Portrait réaliste d'une femme de 25 ans
  • • Cheveux bruns longs
  • • Yeux verts
  • • Maquillage naturel
  • • Qualité photo professionnelle
Problèmes Possibles :
  • • Image déformée : Ajoutez plus de mots au negative prompt
  • • Qualité faible : Augmentez "Sampling Steps" à 40
  • • Erreur CUDA : Réduisez Width/Height à 512x512
  • • Trop lent : Réduisez "Sampling Steps" à 20

CAPTURE D'ÉCRAN À AJOUTER :
Première image générée dans l'interface, avec informations techniques visibles en dessous

4
Sélection et Fixation du Seed

Une fois une image qui vous plaît générée :

  • • Sous l'image générée, cherchez les informations techniques
  • • Notez le numéro "Seed" (exemple : 1234567890)
  • • Dans les paramètres, changez "Seed" de -1 à ce numéro
  • • Désormais, toutes vos générations auront le même visage de base
Importance du Seed :

Le seed contrôle la "base génétique" de votre personnage. En fixant le même seed, vous garantissez que le visage reste cohérent à travers toutes les images du dataset.

NOTEZ CE SEED !

Écrivez ce numéro dans votre carnet physique. C'est la "recette" de votre personnage IA. Si vous le perdez, vous ne pourrez plus reproduire exactement le même visage.
