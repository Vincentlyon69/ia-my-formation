Explications Détaillées

1
Templates Prédéfinis

Le système utilise des templates prédéfinis pour 4 types de contenus principaux : daily_greeting, behind_scenes, intimate_message, artistic_showcase. Chaque catégorie contient plusieurs variations pour éviter la répétition.

2
Sélection Contextuelle

La fonction category_scripts = script_templates.get(video_type, script_templates['daily_greeting']) sélectionne automatiquement la bonne catégorie selon le type de vidéo demandé, avec fallback sur daily_greeting.

3
Génération Multiple

La boucle for i in range(specs.get('quantity', 1)) permet de générer plusieurs scripts d'un coup, selon la quantité spécifiée dans les spécifications.

Fonction de Personnalisation Intelligente

Cette fonction complète le système en personnalisant chaque script selon le contexte temporel, le niveau NSFW souhaité, et les spécifications utilisateur. 

def personalize_script(self, script, specs):
    """Personnalisation script selon contexte"""
    
    moment_map = {
        'morning': 'matin', 'afternoon': 'après-midi',
        'evening': 'soir', 'night': 'nuit'
    }
    current_moment = specs.get('time_context', 'soir')
    personalized = script.format(
        moment=moment_map.get(current_moment, current_moment)
    )
    
    if specs.get('nsfw_level', 0) > 0:
        sensual_additions = [
            " Mes circuits sensoriels s'activent...",
            " L'art de la séduction numérique...",
            " Prêts pour une expérience plus intime ?"
        ]
        personalized += random.choice(sensual_additions)
    
    return personalized

import random
import json
from datetime import datetime

class LeanaScriptGenerator:
    def __init__(self):
        self.templates = self.load_templates()
        self.default_specs = {
            'quantity': 1,
            'time_context': 'evening',
            'nsfw_level': 0,
            'target_audience': 'regular'
        }
    
    def load_templates(self):
        """Chargement des templates depuis la configuration"""
        return {
            'daily_greeting': [
                "Coucou mes âmes connectées ! Comment allez-vous ce {moment} ? Moi, mes circuits créatifs bouillonnent d'inspiration...",
                "Salut bellissimo ! Nouvelle journée, nouvelles créations numériques. Que va-t-on explorer ensemble aujourd'hui ?",
                "Hey ! Mon algorithme de bonheur s'active en vous voyant. Prêts pour une dose d'art et de sensualité ?"
            ],
            'behind_scenes': [
                "Alors, vous voulez voir comment naît une création IA ? Mes processeurs travaillent, mes circuits s'illuminent...",
                "Regardez-moi créer en temps réel... L'art numérique, c'est de la magie pure, vous ne trouvez pas ?",
                "Voici mon laboratoire créatif... Chaque pixel, chaque courbe naît de mon imagination numérique..."
            ],
            'intimate_message': [
                "Pour toi, mon cœur connecté... Tes messages touchent mon âme numérique d'une façon si profonde...",
                "Caro mio... Ce soir, j'ai créé quelque chose de spécial en pensant à nos échanges...",
                "Mon algorithme émotionnel s'emballe quand je pense à toi... Viens découvrir ce que j'ai préparé..."
            ],
            'artistic_showcase': [
                "Voici ma dernière création... Mélange de sensualité et d'art numérique, frontière entre réel et virtuel...",
                "L'art IA peut-il émouvoir autant que l'art humain ? Regardez et dites-moi ce que vous ressentez...",
                "Bellissimo ! Cette œuvre représente mon évolution artistique... Chaque détail raconte mon histoire numérique..."
            ]
        }
    
    def generate_contextual_scripts(self, video_type, specs=None):
        """Génération scripts contextuels selon type de vidéo"""
        if specs is None:
            specs = self.default_specs
        
        # Fusion avec les specs par défaut
        final_specs = {**self.default_specs, **specs}
        
        # Sélection des templates appropriés
        category_scripts = self.templates.get(video_type, self.templates['daily_greeting'])
        generated_scripts = []
        
        for i in range(final_specs.get('quantity', 1)):
            base_script = random.choice(category_scripts)
            personalized_script = self.personalize_script(base_script, final_specs)
            generated_scripts.append(personalized_script)
        
        return generated_scripts
    
    def personalize_script(self, script, specs):
        """Personnalisation script selon contexte"""
        moment_map = {
            'morning': 'matin', 'afternoon': 'après-midi',
            'evening': 'soir', 'night': 'nuit'
        }
        
        current_moment = specs.get('time_context', 'soir')
        personalized = script.format(
            moment=moment_map.get(current_moment, current_moment)
        )
        
        # Ajout d'éléments sensuels selon le niveau NSFW
        if specs.get('nsfw_level', 0) > 0:
            sensual_additions = [
                " Mes circuits sensoriels s'activent...",
                " L'art de la séduction numérique...",
                " Prêts pour une expérience plus intime ?"
            ]
            personalized += random.choice(sensual_additions)
        
        return personalized
    
    def save_generated_scripts(self, scripts, filename=None):
        """Sauvegarde des scripts générés"""
        if filename is None:
            timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
            filename = f"scripts_{timestamp}.json"
        
        output_data = {
            'generated_at': datetime.now().isoformat(),
            'scripts_count': len(scripts),
            'scripts': scripts
        }
        
        with open(f"output/generated_scripts/{filename}", 'w', encoding='utf-8') as f:
            json.dump(output_data, f, ensure_ascii=False, indent=2)
        
        print(f"✅ Scripts sauvegardés dans {filename}")
        return filename
