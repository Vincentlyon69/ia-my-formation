BLOC 3 : ENTRAÎNEMENT MODÈLE LoRA

Créer votre modèle IA personnalisé à partir de votre dataset

Objectif de ce Bloc

Transformer votre dataset d'images (créé dans le Bloc 2) en un modèle LoRA fonctionnel capable de générer fidèlement votre personnage IA féminin dans toutes les situations. À la fin de ce bloc, vous aurez un fichier .safetensors prêt à utiliser dans AUTOMATIC1111.

Pourquoi LoRA plutôt que DreamBooth ?

  • • Plus rapide : 30min-2h vs 4-8h pour DreamBooth
  • • Moins de VRAM : Fonctionne sur GPU gratuits Colab
  • • Plus flexible : Compatible avec tous les modèles de base
  • • Fichier plus petit : 50-200MB vs plusieurs GB

3.2 - Configuration des paramètres d'entraînement

🎯 Paramètres critiques à personnaliser :

INSTANCE_NAME : Mot-clé unique pour reconnaître votre IA

MAX_TRAIN_STEPS : Nombre d'étapes = qualité finale

LEARNING_RATE : Vitesse d'apprentissage = stabilité

# Configuration des paramètres d'entraînement LoRA
print("⚙️ Configuration des paramètres d'entraînement...")

import os

# 🎯 PARAMÈTRES À PERSONNALISER
INSTANCE_NAME = "myiawoman"  # ⚠️ IMPORTANT: Mot unique, pas de caractères spéciaux
INSTANCE_PROMPT = f"a photo of {INSTANCE_NAME} woman"
CLASS_PROMPT = "a photo of a woman"  # Prompt générique pour régularisation

# Chemins des données
INSTANCE_DIR = "/content/drive/MyDrive/IA_MYM_Training_Data/instance_images"
OUTPUT_DIR = "./lora_output"

# Modèle de base (ne pas changer sauf expertise)
MODEL_NAME = "runwayml/stable-diffusion-v1-5"

# Calcul automatique du nombre d'étapes
images_path = INSTANCE_DIR
if os.path.exists(images_path):
    NUM_IMAGES = len([f for f in os.listdir(images_path) 
                     if f.lower().endswith(('.png', '.jpg', '.jpeg'))])
    print(f"📸 {NUM_IMAGES} images détectées dans le dataset")
    
    # Calcul intelligent des étapes
    if NUM_IMAGES < 20:
        STEPS_PER_IMAGE = 150  # Dataset petit = plus d'étapes par image
    elif NUM_IMAGES < 50:
        STEPS_PER_IMAGE = 100  # Dataset moyen
    else:
        STEPS_PER_IMAGE = 80   # Dataset large
        
    MAX_TRAIN_STEPS = STEPS_PER_IMAGE * NUM_IMAGES
    print(f"🔢 Nombre d'étapes calculé: {MAX_TRAIN_STEPS}")
else:
    print("❌ ERREUR: Impossible de compter les images")
    MAX_TRAIN_STEPS = 2000  # Valeur par défaut

# Autres paramètres optimisés
LEARNING_RATE = 1e-4      # Vitesse d'apprentissage stable
TRAIN_BATCH_SIZE = 1      # Batch size pour GPU gratuit
SAVE_INTERVAL = 500       # Sauvegarde tous les 500 steps
RESOLUTION = 512          # Résolution standard

print("✅ Configuration terminée!")
print(f"📋 Résumé des paramètres:")
print(f"   - Token IA: {INSTANCE_NAME}")
print(f"   - Images: {NUM_IMAGES}")
print(f"   - Étapes total: {MAX_TRAIN_STEPS}")
print(f"   - Durée estimée: {MAX_TRAIN_STEPS//100} à {MAX_TRAIN_STEPS//50} minutes")

100 images

# 🚀 LANCEMENT DE L'ENTRAÎNEMENT LORA
print("🚀 Préparation du lancement de l'entraînement...")
print("⏰ NE FERMEZ PAS CET ONGLET PENDANT L'ENTRAÎNEMENT!")

# Création du dossier de sortie
os.makedirs(OUTPUT_DIR, exist_ok=True)

# Construction de la commande d'entraînement
command = f"""
accelerate launch --mixed_precision="fp16" train_dreambooth_lora.py \
  --pretrained_model_name_or_path="{MODEL_NAME}" \
  --instance_data_dir="{INSTANCE_DIR}" \
  --output_dir="{OUTPUT_DIR}" \
  --instance_prompt="{INSTANCE_PROMPT}" \
  --class_prompt="{CLASS_PROMPT}" \
  --resolution={RESOLUTION} \
  --train_batch_size={TRAIN_BATCH_SIZE} \
  --gradient_accumulation_steps=1 \
  --learning_rate={LEARNING_RATE} \
  --lr_scheduler="constant" \
  --lr_warmup_steps=100 \
  --max_train_steps={MAX_TRAIN_STEPS} \
  --checkpointing_steps={SAVE_INTERVAL} \
  --seed=42 \
  --mixed_precision="fp16" \
  --use_8bit_adam \
  --xformers \
  --enable_xformers_memory_efficient_attention \
  --train_text_encoder \
  --rank=32 \
  --with_prior_preservation \
  --prior_loss_weight=1.0 \
  --num_class_images=50
"""

print("💾 Commande d'entraînement préparée")
print("🔥 LANCEMENT DE L'ENTRAÎNEMENT...")
print(f"⏱️ Durée estimée: {MAX_TRAIN_STEPS//100}-{MAX_TRAIN_STEPS//50} minutes")
print("📊 Surveillez les logs ci-dessous...")

# Lancement effectif
!{command}
📊 Comment surveiller l'entraînement :

Logs normaux que vous verrez :

  • • "Loading checkpoint" = Téléchargement du modèle de base
  • • "Training..." = Début effectif de l'entraînement
  • • "Step X/Y, Loss: 0.xxx" = Progression et qualité
  • • "Saved checkpoint" = Sauvegarde intermédiaire

Indicateurs de bonne progression :

  • • Loss qui diminue progressivement (ex: 1.2 → 0.8 → 0.3)
  • • Pas d'erreurs de mémoire (CUDA out of memory)
  • • Sauvegardes régulières toutes les 500 étapes

CAPTURE D'ÉCRAN À AJOUTER :

Console d'entraînement avec :

  • → Logs de progression défilant
  • → Valeurs de "Loss" et "Step"
  • → Messages de sauvegarde
  • → Barre de progression si visible
🔧 Problèmes pendant l'entraînement

"CUDA out of memory" :

  • → Réduire TRAIN_BATCH_SIZE à 1
  • → Réduire RESOLUTION à 384
  • → Redémarrer l'environnement

Session Colab interrompue :

  • → Reprendre depuis le dernier checkpoint
  • → Réduire MAX_TRAIN_STEPS
  • → Considérer Colab Pro

Loss qui ne diminue pas :

  • → Vérifier la qualité du dataset
  • → Réduire le LEARNING_RATE
  • → Augmenter le nombre d'étapes

4.2 - Test rapide de génération

💡 Pourquoi tester ? Avant de télécharger le modèle, testons s'il fonctionne et génère bien votre personnage IA. Cela évite de perdre du temps avec un modèle défaillant. 

# Test rapide de génération avec le modèle LoRA entraîné
print("🎨 Test de génération avec votre modèle LoRA...")

from diffusers import StableDiffusionPipeline
import torch

# Chargement du pipeline de base
print("📥 Chargement du modèle de base...")
pipe = StableDiffusionPipeline.from_pretrained(
    MODEL_NAME,
    torch_dtype=torch.float16,
    safety_checker=None,
    requires_safety_checker=False
)

# Chargement du LoRA entraîné
lora_path = f"{OUTPUT_DIR}/pytorch_lora_weights.safetensors"
if os.path.exists(lora_path):
    print("🔗 Chargement de votre LoRA personnalisé...")
    pipe.load_lora_weights(OUTPUT_DIR)
    pipe = pipe.to("cuda")
    
    # Prompts de test
    test_prompts = [
        f"a photo of {INSTANCE_NAME} woman, portrait, high quality",
        f"a photo of {INSTANCE_NAME} woman, smiling, natural lighting",
        f"a photo of {INSTANCE_NAME} woman, elegant dress, professional photo"
    ]
    
    print("🎨 Génération d'images de test...")
    for i, prompt in enumerate(test_prompts):
        print(f"   Test {i+1}/3: {prompt}")
        
        # Génération
        with torch.autocast("cuda"):
            image = pipe(
                prompt,
                negative_prompt="ugly, deformed, blurry, bad anatomy",
                num_inference_steps=30,
                guidance_scale=7,
                width=512,
                height=768
            ).images[0]
        
        # Sauvegarde du test
        test_filename = f"test_generation_{i+1}.png"
        image.save(test_filename)
        print(f"   ✅ Sauvegardé: {test_filename}")
    
    print("🎉 Tests terminés! Vérifiez les images générées:")
    print("   - Le visage correspond-il à votre dataset ?")
    print("   - La qualité est-elle satisfaisante ?")
    print("   - Les variations fonctionnent-elles ?")
    
else:
    print("❌ Fichier LoRA non trouvé - Impossible de tester")

4.3.1 - Prompts optimisés pour MYM

🎯 Structure de prompt recommandée :
<lora:IA_MYM_LoRA_v1:0.8> a photo of myiawoman woman, [DESCRIPTION], [STYLE], [QUALITY]

Où :
- [DESCRIPTION] = Description de la pose, expression, vêtements
- [STYLE] = Style photographique (portrait, boudoir, artistic, etc.)
- [QUALITY] = Tags de qualité (8k, high quality, photorealistic, detailed)
✅ Prompts SFW Efficaces
<lora:IA_MYM_LoRA_v1:0.8> a photo of myiawoman woman, elegant dress, confident smile, professional portrait, studio lighting, 8k, high quality
<lora:IA_MYM_LoRA_v1:0.8> a photo of myiawoman woman, casual outfit, natural expression, outdoor setting, golden hour lighting, photorealistic
🔥 Prompts NSFW (Privé uniquement)
<lora:IA_MYM_LoRA_v1:0.8> a photo of myiawoman woman, black lingerie, seductive pose, bedroom setting, soft lighting, detailed, 8k
<lora:IA_MYM_LoRA_v1:0.8> a photo of myiawoman woman, artistic nude, elegant pose, professional photography, dramatic lighting
⚠️ Negative Prompts Essentiels
ugly, deformed, blurry, bad anatomy, extra limbs, missing limbs, bad hands, bad fingers, extra fingers, missing fingers, watermark, signature, text, logo, low quality, artifacts, jpeg artifacts

Copiez toujours ce negative prompt pour éviter les déformations communes
